Image

To display an image in HTML we use the img element. Unlike the other elements we have encountered, the img element is self-closing. Empth, self-closing HTML elements do not need a closing tag.

Instead of wrapping content with an opening and closing tag, it embeds an image into the page using a src attribute which tells the browser where the image file is located.

Ic can embed an image using both absolute and relative paths.

absolute path / https://www.theodinproject.com/mstile-310x310.png

The alt attribute is used to describe an Image. It will be used in place of the image if it cannot be loaded. It is also used with screen readers to describe shat the image is to visually impaired users.
